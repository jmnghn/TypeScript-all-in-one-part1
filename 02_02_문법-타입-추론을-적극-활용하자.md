## 타입 추론을 적극 활용하자

any가 아닌 이상 추론이 잘 되면, 추론에게 맡기자는 개인적인 생각.<br />

```ts
function add(x: number, y: number) {
  return x + y;
}
const result = add(1, 2); // const result: number
```

```ts
const arr = ["1", "2", "3"]; // const arr: string[]
```

```ts
const arr = [1, 2, 3]; // const arr: number[]
```
