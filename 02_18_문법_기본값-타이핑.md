## 기본값 타이핑

타이핑이 되고나면 헷갈리는 몇가지... ^^;

### 1

```ts
// ...이게 뭐더라? 😅 타입이 붙고나면 헷갈리는 인자 기본 값
const a = (b: number = 3, c: number = 5) => {
  return "3";
};
```

<br />

### 2

```ts
// 객체도 마찬가지 ^^;
const a = (b: { c: string } = { c: "d" }) => {};
```

<br />

### 3

```ts
// JSX에서의 <> 로 error를 방지하기 위한...
const add = <T = unknown>(x: T, y: T): T => ({ x, y });
const add = <T extends unknown>(x: T, y: T): T => ({ x, y });
```
